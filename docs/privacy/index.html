AUFAIT Enterprise Risk Management (ERM) – Privacy Policy

Version 1.0 · Last Updated: October 2025

1. Overview

AUFAIT Enterprise Risk Management (ERM) is a SharePoint Framework (SPFx)–based solution developed by Aufait Technologies Pvt. Ltd. It helps organizations identify, assess, and manage risks directly within their Microsoft 365 environment. This Privacy Policy describes how AUFAIT ERM handles information within the customer’s Microsoft 365 tenant.

2. Data Storage and Processing

AUFAIT ERM operates entirely within the customer’s Microsoft 365 / SharePoint Online tenant. The solution does not transmit, store, or process any data outside of the customer’s tenant environment. All information, including user references (such as assigned risk owners, departments, and roles), remains within the SharePoint lists configured for the solution.

No data is collected or stored on external servers owned or operated by Aufait Technologies. All authentication and access control are managed through Azure Active Directory (Microsoft Identity Platform).

3. Types of Data Handled

AUFAIT ERM only stores limited user-related metadata within SharePoint lists, such as:

  • User display names or assigned risk owners
  • Department or project associations
  • Workflow activity and approval history

The app does not collect or process sensitive categories of data such as racial origin, political opinions, religious beliefs, genetic or biometric information, or health data.

4. Data Access, Modification, and Deletion

AUFAIT ERM provides site administrators full control over data stored within the app’s SharePoint lists. Administrators can:

  • Delete user-related records (e.g., remove a person from a risk entry, contingency plan, or role list)
  • Update user and role assignments within the lists
  • Restrict or limit processing by removing or deactivating user entries

These operations affect only the data maintained by AUFAIT ERM inside SharePoint. The app does not delete or modify any Microsoft 365 user accounts, profiles, or tenant-level information.

5. Data Retention

Data created by AUFAIT ERM remains within the customer’s SharePoint site until manually deleted or archived by the site administrators. The retention period and deletion policies are controlled by the organization’s own Microsoft 365 data governance and compliance settings.

6. Data Security

AUFAIT ERM relies on the enterprise-grade security provided by Microsoft 365. Authentication, access control, and data protection are managed by Azure Active Directory and Microsoft’s built-in Conditional Access policies. The solution does not store passwords, tokens, or credentials in code.

7. Compliance

AUFAIT ERM follows the compliance and privacy framework of Microsoft 365, which includes certifications for GDPR, ISO 27001, ISO 27018, and related data protection standards. Aufait Technologies does not process customer data outside the Microsoft 365 platform.

8. Automated Processing and Analytics

AUFAIT ERM does not perform any automated decision-making, user profiling, or analytics involving personal data. Workflow automation is limited to Power Automate processes explicitly configured by the customer.

9. Children’s Data

AUFAIT ERM is intended for enterprise and organizational use only and does not collect or process information from individuals under the age of 16.

10. Privacy Reviews

Aufait Technologies performs internal privacy and security reviews for each major release of AUFAIT ERM to ensure ongoing compliance and secure handling of information.
