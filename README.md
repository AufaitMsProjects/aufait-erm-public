**AUFAIT Enterprise Risk Management (ERM)** is a SharePoint Framework (SPFx)-based solution designed to help organizations **identify, assess, mitigate, and monitor enterprise risks** within Microsoft 365.  

It integrates seamlessly with **SharePoint Online**, **Power Automate**, and **Power BI** to deliver a unified risk governance experience.

**Key Features**
- Centralized SharePoint-based risk management hub  
- Automated mitigation and contingency workflows via Power Automate  
- Configurable user roles, permissions, and branding  
- Detailed audit trails for compliance  
- Power BI-ready data structure for analytics
